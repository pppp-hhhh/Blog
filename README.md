# [Blog](https://wuqu-sooty.vercel.app/)

## 技术栈

- Nuxt 4 + Vue 3
- Nuxt UI + TailwindCSS v4
- Nuxt Content
- SQLite (better-sqlite3)
- Vercel (KV 存储)

## 安装方法

### 环境要求

- Node.js 18+
- pnpm 9.0+

### 安装步骤

1. **克隆项目**

```bash
git clone https://github.com/pppp-hhhh/Blog
cd Blog
```

2. **安装依赖**

```bash
pnpm install
```

3.**启动开发服务器**

```bash
pnpm dev
```

访问 `http://localhost:3000`
